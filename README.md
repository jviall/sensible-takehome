# sensible-takehome
Basic integration with [Google Maps's "Nearby Search" API](https://developers.google.com/maps/documentation/places/web-service/search-nearby)

## Init
> Requires Node 18+ & Yarn
```
yarn
```

## Run
```
yarn dev
```

## Preview Production
```
yarn build && yarn Preview
```

## Built with
* React 18 + Suspense
* TypeScript
* Vite + SWC
* Tailwind CSS
* React Query
* Google Maps API
